### 🧠 Về tôi (Hồ sơ năng lực thực tế)
- ☕ **Thành phần cấu tạo:** 70% Cà phê, 25% Nhạc Remix (để bay), 4% Nhạc Thiền (để cứu rỗi linh hồn khi chuẩn bị đập máy), và 1% xem các idol "lắc đít" trên TikTok để giữ thăng bằng cuộc sống.
- 🤖 **Workflow thời đại AI:** Đưa prompt cho AI -> Nhấn run -> Ngồi lướt TikTok chill chill chờ bug tự fix (hoặc tự sinh ra).
- 💀 **Chiến tích huy hoàng:** Đã từng dũng cảm xóa sạch file cấu hình `nginx config` ngay trên **Production** rồi tự tin gõ `nginx -s reload`. Hôm đó hệ thống không chạy, nhưng tim tôi thì chạy với tốc độ 300km/h.
